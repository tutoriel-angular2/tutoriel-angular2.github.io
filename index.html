Bienvenu sur le Tutoriel angular 2

IL y a déjà plusieurs mois les developpeurs de Angular chez Google annoncait l'arrivée de Angular 2 en version stable. Il y a une polemique grande autour du nouveau Framework qui était selon certains trop compliqué à prendre en main notamment avec l'introduction de Typescript comme langage recommandé pour écrire les applications angular 2.
Aujourd'hui Angular 2 est stable et prêt a être utilisé, on est heureux de vous servir ce grand tutoriel angular2 qui s'inscrit comme un des premiers dans la communauté de developpeurs francais.
Le tutoriel part de la grande introduction au nouveau Framework développé par Google à une entrée dans les concepts avancés jusqu'à la mise en production
!!!!Le tutoriel est en cours de rédaction!!!!
Merci de suggerer vos corrections et suggestions sur notre GitHub Repository

Introduction à Angular 2 et installation des outils
C'est quoi angular 2? Qui y a t'il de nouveau dans le framework? Dans ce chapitre on procedera a une introduction globale à Angular 2! Aucune experience avec angularJS 1.x n'est requise même si quelques fois des petites comparaisons seront faites entre les deux versions du framework.
2

Ma première application angular 2
Après avoir installé les outils, on developpera notre toute premiere application angular qui ne fera rien d'autre qu'afficher le contenu d'une variable dans nos vues.
3

Typescript, prendre le langage en main
Typescript est le langage recommandé quand on veut developper des applications angular 2. Ce chapitre introduit Typescript de facon tres simple et brève vu que Typescript est tres proche du javascript traditionnel.
4

Comment Angular 2 marche ?
Après les premiers pas dans le langage, on va entrer plus profondement dans le langage en comprenant les concepts de base et comment toutes les composantes fusionnent ensemble
5

Les directives incluses dans le framework
Angular 2 est livré dans son core avec des directives prédefinies. Dans ce chapitre chacune de ces directive introduite avec des exemples d'utilisation
6

Les Formulaires avec Angular 2
Les formulaires representent le point de contact entre les utilisateurs et vos applications. Dans ce chapitres on apprendra comment creer, valider et gerer des formulaires avec angular 2
7

HTTP, Parlez avec vos APi
Aujourd'hui on ne parle plus que WebService, dans ce chapitre on introduira un des modules fondamentaux de angular 2 qui est le composant HTTP. on fera des CRUD operations sur des APIs
8

Le Routing avec Angular 2
Dans ce chapitre on apprendra comment naviguer dans l'application et comment transporter les informations d'une vue a une autre
9

L'Injection de dependances
L'injection des dependances fait partie des notions les plus importantes dans les frameworks actuels sur le marché et angular 2 n'y echappe pas. Angular 2 a dans son coeur une veritable machine qui gere les dependances entre les differentes composantes de notre application.
10

L'architecture des données dans une application angular 2
Gerer les données dans une application peut s'averer tres complexe des que celle-ci grandit. Dans ce chapitre on apprendra a manager les données de notre applications obtenues de differentes sources (RestCall, LocalStorage, ...)
11

Les observables
Qui parle de données parlent de changement! Comment s'y prendre avec les données qui changent ? Ce chapitre est la reponse à cette problematique.

Introduction à Redux
Les UIs deviennent de plus en plus complexes et dans les SPA (Single Page App) on a besoin de gerer l'etat de la route active, d'un tab qu'on a selectionné, on a besoin d'afficher un SPinner quand les données chargent ou quand un calcul est en cours.

Quelques composants avancées

Tester une application Angular 2

Angular 2 en production

Angular 2 dans AngularJs 1
Vous avez une tres grosse application angularjs 1.x et souhaitez quand meme introduire angular 2 dans celle-ci ? Ce chapitre nous apprendra comment ca se fait
